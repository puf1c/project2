1. Какие графики лучше всего подходят для визуализации погодных данных?

Я считаю что в нашем случае для всех параметров лучше всего подходит обычная гистограмма, так как данные простые и не нуждаются в сложной визуализации. Гистограмма позволяет пользователю легко и наглядно сравнить показатели для разных городов.
Но в общем для описания температуры лучше всего подходят столбчатая диаграмма(удобно сравнивать температуру в разных точках) и линейный график(хорошо заметны тенденции изменения)
для описания Влажности лучше использовать гистограмму(наглядная визуализация распределения) или точечный график(для анализа взаимосвязи с другими параметрами)
Скорость ветра и вероятность осадков можно описывать столбчатой диаграммой, так как она хороша для сравнения показателей между собой
Также для вероятности осадков в редких случаях можно использовать круговую диаграмму для того чтобы визуализировать соотношение разных величин вероятности (например за год)

2. Во-первых, можно добавить возможность выбора параметров для отображения с помощью чекбоксов, это значительно упростит визуализацию и не будет перегружать график.
Во-вторых, стоит использовать динамическую фильтрацию данных (то есть ползунки для выбора интервала значений), что позволит пользователю оставить на графике только актуальные для него данные, что может упростить анализ.
В-третьих, внедрить интерактивные всплывающие подсказки при наведении курсора, то есть сразу получить конкретные данные без дополнительной настройки.
Также при необходимости можно применять анимацию в визуализации данных, чтобы наглядно показать изменение величины с течением времени.
